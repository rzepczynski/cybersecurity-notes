## Projekty

### Windows Log Analysis (SOC Simulation)

Symulacja podejrzanej aktywności w systemie Windows oraz analiza logów bezpieczeństwa z wykorzystaniem Event Viewer.

Projekt obejmuje:
- tworzenie użytkownika
- nadawanie uprawnień administratora
- analizę Event ID 4720, 4732 i 4624
